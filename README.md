# Pawfection 
Pawfection is our final-year college project, developed as a full-stack web application to help dog and cat owners in Ireland manage their pets’ care in one central platform. The system allows users to create pet profiles, manage appointments and reminders, track pet-related items, and use a Lost & Found feature to report missing pets or submit sightings. The project also includes Basic, Premium, and Admin user roles, with Premium features such as Google Maps location tracking and an AI Pet Assistant. Pawfection was built using React for the frontend, Laravel for the backend, and MySQL for the database. Overall, the project shows how technology can make pet care more organised, improve communication during lost-pet situations, and support responsible pet ownership.
